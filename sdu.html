SDU 教學

找山數 → 方向 → 離場點 → 進場點

SDU 範例（鏡射 SDD）：方向 2H · 弟弟 1H · 哥哥 4H · 離場 8H OWD

← 回首頁

1

找方向（最多山數）

  • 起點：從距離 K 棒歷史最新的橙色破 0 軸開始。
  • 時間軸：以 1H 為起點，依序查找：1H → 90 分 → 2H → 3H → 4H → 6H → 8H → 12H → 1D，直到找到方向（最多山數）。
2

確認方向與弟弟同陣線

  1. 假設在 2H 找到方向，身份 SDU。
  2. 弟弟時間軸 1H，確認也是 SDU。
  3. 方向與弟弟同陣線 → 繼續。
3

找哥哥身份並確認 OBV 同陣線

  1. 哥哥 4H SDU；哥哥的哥哥 8H OWD。
  2. 4H OBV 多頭縮量，與 2H 方向 OBV 同陣線。
4

確定離場點位置

  1. 8H 身份 OWD。
  2. 離場點 = 哥哥的哥哥 8H 的 EMA52。

至此已確認方向與離場點。

5

驗證離場點位置是否準確

要驗證「離場點」是否可靠，需完成以下三步驟（共 3 個驗證）：

5.1 在 8H 畫離場水平線

  • 在哥哥的哥哥 8H 圖上，於 EMA52 處畫出水平線，標記離場點。

5.2 回 2H 畫 FIBO（SDU 離場點，鏡射＝SDD 進場畫法）

  • 取相對高點（A）與相對低點（B）：A 點左上→B 點右下 ↘↑，或 A 點右上→B 點左下 ↑↙。
  • 檢查 FIBO 0 / 0.382 / 0.5 / 0.618 是否與 8H EMA52 水平線重疊。

5.3 回 8H 畫成交量分佈圖

  • 在 EMA52 附近畫波段成交量分佈圖，檢查 POC 是否與 EMA52 重疊 → 第三個驗證。

驗證小結（離場點）

驗證內容
1EMA52 水平線（離場點位置）
22H FIBO 0 / 0.382 / 0.5 / 0.618 與 EMA52 重疊
38H 波段成交量分佈圖 POC 與 EMA52 重疊

三項皆符合 → 離場點驗證完成。

6

找進場點位置

以下在 2H 時間軸（方向時間軸）進行。

6.1 畫第一個 FIBO（SDU 進場點，鏡射＝SDD 離場畫法）

  • 相對低點與高點：A 點左下→B 點右上 ↗↓，或 A 點右下→B 點左上 ↓↖。

6.2 標記 0.618、1.382

  • 0.618 畫水平線 → 可先進一半倉；1.382 畫水平線作進場目標。

6.3 第二個 FIBO 找重疊

  • 再畫第二個 FIBO，看是否與 0.618、1.382、1.618 重疊；目標至少兩條線重疊。

6.4 4H 支撐與阻力確認

  • 切到 4H，檢查進場重疊水平線是否落在支撐或阻力上；保留該區，其餘可清除。

進場點證據小結

證據內容
14H 支撐與阻力位置
2兩個 FIBO 重疊（0.618 / 1.382 / 1.618）
3進場水平線落在支撐或阻力上
7

檢查 SDU 是否會漲到進場點位置

如何判斷 2H 方向 SDU 會漲到進場點？看 4H OBV：

4H OBV 狀態解讀進場點建議
OBV > 0，多頭縮量仍有一口氣會漲上去，碰到進場區後再考慮離場。進場點維持在原本重疊區。
OBV < 0沒有氣了，上漲力道不足。進場點改在 0.618 位置（更保守）。
→
